Advance CSS Notes:-
1. Box-Shadow - To make a Shadow of a box

2. Text-Shadow - To make a shadow of any text

3. Conic-Gradient - It's a type of gradient where color will mixing like a bold color or a gradient type.

4. @keyframes - To animate something
@keyframes{
    from{
        background-color: red;
        border-radius: 5px;
    }
    to{
        background-color: blue;
        border-radius: 50px;
    }
}

5. @property - Use for making custom variables
@property --bgColor{
    syntax: "<color>"; //color, length, angle
    initial-value: red;
    inherit: false;
}

6. :has() - It uses for transform a hover effect to opposite at same time
    #nav:has(.link:hover) .link {
      transform: scale(0.7);
      opacity: 0.4;
    }

    #nav .link:hover {
      anchor-name: --some-anchor;
      color: rgb(200, 255, 0);
      font-weight: 900;
      transform: scale(1.1) !important;
      /*Use scale higher than 1 because it causing hover issues...*/
      opacity: 1 !important;
    }
